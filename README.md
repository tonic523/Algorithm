## 풀이한 문제 작성 방법
- 만약 문제 폴더가 없거나 추가로 넣고 싶으면 홈에서 Add file -> Create New File 을 누른다.
- 파일 명에 문제이름/README.md 를 작성하면 폴더를 생성하면서 README.md를 수정할 수 있다.
- README.md 에 풀이한 방법, 과정을 작성한다.
- 풀이한 코드는 ps1, ps2, ... 로 풀어본 횟수에 맞게 .py파일로 추가한다.
