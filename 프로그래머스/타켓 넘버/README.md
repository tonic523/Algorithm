## 풀이1
- 성공

> DFS

처음엔 같은 숫자만 있다는 가정하에 조합으로 풀이하면 될 줄 알았지만\
역시 문제는 꼼꼼하게 읽어봐야 합니다 ㅎㅎ\
스택 자료구조로 DFS를 구현하여 풀이했습니다.\
- 숫자 배열 중 첫 번째 값을 가져오고 두 번째 값을 더한 것과 뺀 값 그리고 몇번 더했는지를 튜플 값으로 스택에 넣습니다.
- 더한 개수가 배열의 길이가 되고 더한 값이 목표가 되면 결과값에 1을 더합니다.
- 스택에 남은 값이 없을 때까지 반복!
